# beat-track predictions

_Pre-analysis predictions from Manning, written before any substantial listening-history import._

The point of this file is to avoid fake hindsight. Predictions go here first; later analysis can compare against them.

## predictions

1. **Strong return gravity**
   - Rich is unlikely to be pure novelty-churn listener.
   - Expect a set of artists/albums that recur persistently over long periods.

2. **Multiple stable taste zones**
   - Taste is unlikely to collapse into one neat genre blob.
   - More likely: a few recurring identity zones with different moods/textures.

3. **Time/context-sensitive listening**
   - Expect meaningful shifts by time of day, work mode, season, or context.
   - Some listening modes will likely be exploratory, others more comfort/repetition based.

4. **Texture and affect > coarse genre labels**
   - Genre labels will probably explain less than mood, density, warmth, sharpness, melancholy, drift, intensity, or similar texture variables.

5. **Repetition spikes tied to projects or headspace**
   - Expect periods of fixation or concentrated repeat listening.
   - Those periods may correlate with project phases, seasonal cycles, or specific emotional states.

6. **Technical neatness mixed with emotional weather**
   - Predicted taste shape: not random eclecticism, not purely cerebral curation either.
   - Likely a mix of structured/intentional choices and atmosphere-driven or emotionally resonant returns.
